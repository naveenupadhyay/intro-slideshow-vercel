# Intro Slideshow Vercel Project

Standalone Vite/React extraction of the EleventyfirstParallel AI intro slideshow.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Vercel

Import this folder as a new Vercel project.

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
- Install command: `npm install`

After the first deployment, add the new domain in Vercel under Project Settings > Domains.
